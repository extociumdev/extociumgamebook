# 💳 Asset Rental

{% tabs %}
{% tab title="ENG" %}
### 🧩 Asset Rental

Asset Rental allows you to\
**temporarily rent Heroes and Weapons** for a set period of time.

Through rentals,\
you can try out Heroes or Weapons you do not own\
and experience their performance and playstyle in actual combat.

***

#### ◾ Asset Rental Content Overview

Within the Asset Rental menu, you can access the following rental services:

**Hero Rental**\
👇 Use Gems to rent a Hero and play with the rented Hero for a limited time

{% content-ref url="hero-rental.md" %}
[hero-rental.md](hero-rental.md)
{% endcontent-ref %}

**Weapon Rental**\
👇 Weapon rental service (Coming Soon)

{% content-ref url="weapon-rental.md" %}
[weapon-rental.md](weapon-rental.md)
{% endcontent-ref %}

Detailed instructions for each rental service can be found on their respective guide pages.

***

✨

> **Use Asset Rental to explore different Heroes and Weapons**\
> **and find the ones that fit your playstyle.**

### 🧩 자산 렌탈 (Asset Rental)

자산 렌탈은 **영웅과 무기를 일정 기간 동안 대여하여 사용할 수 있는 콘텐츠**입니다.

전투 스타일과 성능을 미리 체험할 수 있습니다.

### 🧩 アセットレンタル（Asset Rental）

アセットレンタルは、\
