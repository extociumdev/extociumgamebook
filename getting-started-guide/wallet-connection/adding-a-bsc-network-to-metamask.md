❗[When installing the wallet, be sure to write down the recovery phrase on paper and store it in a place where it won't be lost!](precautions.md#eng-1)

***

> 🦊All NFTs and tokens used in EXTOCIUM are based on the Binance Smart Chain (BSC). Therefore, to fully enjoy the game, you need to add the BSC network to MetaMask.

2️⃣**Access the MetaMask app and try touching (clicking) the area circled in red in the image below.**

<figure><img src="../../.gitbook/assets/KakaoTalk_20231016_130658726_02 (1).jpg" alt="" width="375"><figcaption><p>Touch the "Wallet"</p></figcaption></figure>

{% hint style="info" %}
The red circled area is where you set the 'Network' in MetaMask. The default setting of MetaMask is the 'Ethereum Network'. When using EXTOCIUM, it must always be set to the BSC network.
{% endhint %}

3️⃣**Touch (click) the 'Add Network' button marked with a check in the image below.**

<figure><img src="../../.gitbook/assets/KakaoTalk_20231016_130658726_01.jpg" alt="" width="375"><figcaption><p>Add Network - Binance Chain</p></figcaption></figure>

**4️⃣From the list, select 'Binance Smart Chain' or 'BNB Smart Chain'.**

<figure><img src="../../.gitbook/assets/image (618).png" alt="" width="306"><figcaption></figcaption></figure>

**👉**Simple, right? The addition of the BSC network is now complete!**🎉**

<figure><img src="../../.gitbook/assets/image (617).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
The area underlined at the top of the wallet should be set to BNB Smart Chain or Binance Smart Chain, as shown in the image above!
{% endhint %}

***

❓**If the network does not exist, please follow the guide below.**

<figure><img src="../../.gitbook/assets/KakaoTalk_20231016_130658726.jpg" alt="" width="188"><figcaption><p>Custom Network</p></figcaption></figure>



**After touching the 'Add Network' button**, select the '**Custom Networks**' menu in the top right corner, enter the following information into each field, and touch (click) the '**Add**' button to complete the registration!

&#x20;                                              👉Network Name : **Binance Smart Chain**\
&#x20;                                              **👉**RPC URL : [**https://bsc-dataseed.binance.org**](https://bsc-dataseed.binance.org/)\
&#x20;                                              **👉**Chain ID : **56**\
&#x20;                                              **👉**Symbol : **BNB**\
&#x20;                                              **👉**Block Explorer URL : [**https://bscscan.com**](https://bscscan.com/)
{% endtab %}

***

> 🦊EXTOCIUM 게임에서 사용되는 NFT 및 토큰은 모두 바이낸스 스마트 체인 (BSC)를 기반으로 합니다. 따라서 게임을 완전하게 즐기려면 메타마스크에 BSC 네트워크를 추가해야 합니다.&#x20;

2️⃣**메타마스크 앱에 접속하여** **아래 이미지의 빨간 동그라미 부분을 터치(클릭) 해 보세요.**&#x20;

<figure><img src="../../.gitbook/assets/KakaoTalk_20231016_130658726_02 (1).jpg" alt="" width="375"><figcaption><p>Touch the "Wallet"</p></figcaption></figure>

{% hint style="info" %}
빨간 동그라미 영역이 메타마스크에서 '네트워크'를 설정하는 영역입니다. 메타마스크의 초기 설정값은 '이더리움 네트워크' 입니다. EXTOCIUM을 사용할 때는 항상 BSC 네트워크로 세팅 되어 있어야 합니다.
{% endhint %}

3️⃣**아래 이미지에 체크 표시된 ‘네트워크 추가’ 버튼을 터치(클릭)하세요.**

<figure><img src="../../.gitbook/assets/KakaoTalk_20231016_130658726_01.jpg" alt="" width="375"><figcaption><p>Add Network - Binance Chain</p></figcaption></figure>

**4️⃣목록에서 'Binance Smart Chain' 혹은 'BNB Smart Chain'을 선택하세요.**&#x20;

<figure><img src="../../.gitbook/assets/image (618).png" alt="" width="306"><figcaption></figcaption></figure>

**👉**간단하죠? BSC 네트워크 추가가 완료되었습니다!**🎉**

<figure><img src="../../.gitbook/assets/image (617).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
지갑 상단에 밑줄 친 영역이 위 이미지와 같이 BNB Smart Chain 혹은 Binance Smart Chain으로 설정되어 있으면 됩니다!
{% endhint %}

***

❓**만약 네트워크가 존재하지 않을 경우, 아래 가이드를 따라해 주세요.**

***

> 🦊EXTOCIUMで使用されるすべてのNFTおよびトークンは、バイナンススマートチェーン（BSC）を基盤としています。そのため、ゲームを完全に楽しむには、MetaMaskにBSCネットワークを追加する必要があります。

2️⃣**MetaMaskアプリにアクセスして、以下の画像の赤い円で囲まれたエリアをタッチ（クリック）してみてください。**

<figure><img src="../../.gitbook/assets/KakaoTalk_20231016_130658726_02 (1).jpg" alt="" width="375"><figcaption><p>Touch the "Wallet"</p></figcaption></figure>

{% hint style="info" %}
赤い円で囲まれたエリアはMetaMaskで「ネットワーク」を設定する場所です。MetaMaskの初期設定値は「イーサリアムネットワーク」です。EXTOCIUMを使用するときは、常にBSCネットワークに設定されている必要があります。
{% endhint %}

3️⃣**下記の画像にチェックマークされた「ネットワーク追加」ボタンをタッチ（クリック）してください。**

<figure><img src="../../.gitbook/assets/KakaoTalk_20231016_130658726_01.jpg" alt="" width="375"><figcaption><p>Add Network - Binance Chain</p></figcaption></figure>

**4️⃣リストから「Binance Smart Chain」または「BNB Smart Chain」を選択してください。**

<figure><img src="../../.gitbook/assets/image (618).png" alt="" width="306"><figcaption></figcaption></figure>

**👉**簡単でしょう？ BSCネットワークの追加が完了しました！**🎉**

<figure><img src="../../.gitbook/assets/image (617).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
ウォレットの上部に下線が引かれたエリアが、上記の画像のようにBNB Smart ChainまたはBinance Smart Chainに設定されていればOKです！
{% endhint %}

***
