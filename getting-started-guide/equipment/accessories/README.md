### ✨ **악세서리 – 영웅의 힘을 극대화하라!** ✨

목걸이, 팔찌, 귀걸이로 구성된 악세서리는 영웅의 능력을 더욱 강력하게 강화합니다.\
각 악세서리는 특정 능력치를 보강하며, 전투에서 더욱 유리한 위치를 차지할 수 있도록 도와줍니다.

🎯 **적절한 악세서리를 조합하여 최고의 전투력을 갖춘 영웅을 완성하세요!** ⚔️🔥
