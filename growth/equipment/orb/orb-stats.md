# ✨ Orb Stats

{% tabs %}
{% tab title="ENG" %}
### 💡 Orb Stats

When an Orb is created, stat options are applied based on the conditions below.\
Additionally, enchanting an Orb grants **extra options depending on the Enchant level**.

***

#### ◾ Orb Stat Option Structure

An Orb may have the following stat options applied:

* [Fixed Options](orb-stats.md#id-1-fixed-options)
* [Primary Stat Options (Chance-based)](orb-stats.md#id-2-primary-stat-options-chance-based)
* [Secondary Stat Options (Chance-based)](orb-stats.md#id-3-secondary-stat-options-chance-based)
* [Elemental Affinity Options (Chance-based)](orb-stats.md#id-4-elemental-affinity-options-chance-based)
* [Enchant Options based on Enchant Level](orb-stats.md#id-5)

{% hint style="info" %}
⚠️ Minimum and maximum values of options may be adjusted\
to **percentage-based values** depending on balance changes.
{% endhint %}

***

#### 1️⃣ Fixed Options

Basic options that are applied to **all Orbs at 100%**.

* Randomly applied within a predefined **minimum–maximum range**
* Fixed option values vary depending on the **Orb type**

| Option             | Spawn Probability (%) |
| ------------------ | --------------------- |
| Evasion Rate (EVA) | 100                   |
| HP                 | 100                   |

💡 **Fixed Option Range Guide**\
👉 [You can check the minimum–maximum values of fixed stats by equipment type.](orb-stats.md#fixed-option-range)

***

#### 2️⃣ Primary Stat Options (Chance-based)

Stat options that are applied **probabilistically** when an Orb is created.

* Application chance varies by Orb
* Randomly applied within a fixed **minimum–maximum range**
* Higher Orb levels increase the chance of option generation
* Each option is applied with an **independent probability**

| Option | Spawn Probability (%) |
| ------ | --------------------- |
| STR    | 5\~8                  |
| DEX    | 5\~8                  |
| INT    | 5\~8                  |
| CON    | 5\~8                  |
| WIZ    | 5\~8                  |
| CHA    | 5\~8                  |

💡 **Primary Stat Option Range Guide**\
👉 [Check the minimum–maximum values for Primary Stat Options.](orb-stats.md#primary-stat-option-range)

***

#### 3️⃣ Secondary Stat Options (Chance-based)

Additional stat options that may be applied **probabilistically** when an Orb is created.

* Application chance varies by Orb
* Randomly applied within a fixed **minimum–maximum range**
* Higher Orb levels increase the chance of option generation
* Each option is applied with an **independent probability**

| Option                          | Spawn Probability (%) |
| ------------------------------- | --------------------- |
| Phys. DMG                       | 8\~10                 |
| HP                              | 8\~10                 |
| MP                              | 8\~10                 |
| Phys. Crit DMG                  | 8\~10                 |
| Phys. Crit %                    | 3                     |
| Mag. DMG                        | 8\~10                 |
| Mag. Crit DMG                   | 8\~10                 |
| Mag. Crit %                     | 3                     |
| Reduces skill reload time (CDR) | 4                     |
| Evasion Rate (EVA)              | 0.5\~1                |

💡 **Secondary Stat Option Range Guide**\
👉 [Check the minimum–maximum values for Secondary Stat Options.](orb-stats.md#secondary-stat-option-range)

***

#### 4️⃣ Elemental Affinity Options (Chance-based)

When an Orb is created, an **Elemental Bonus Damage option**\
based on elemental affinity may be applied probabilistically.

<figure><img src="../../../.gitbook/assets/image (1171).png" alt=""><figcaption></figcaption></figure>

* Application chance varies by Orb
* Randomly applied within a fixed **minimum–maximum range**
* Higher Orb levels increase the chance of option generation

{% hint style="info" %}
The additional damage option according to compatibility is interpreted as follows:

<img src="../../../.gitbook/assets/image (621).png" alt="" data-size="original">\
This indication means that it inflicts 2% additional damage to opponents (both PCs and NPCs) that have the 'Light Attribute'. If the final damage I deal to an enemy is 200, then when the enemy has the 'Light Attribute', the final damage becomes 204.

❓<mark style="color:blue;">**200\*(1+0.02)=204**</mark>
{% endhint %}

***

#### 5️⃣ Enchant Options

When an Orb is enchanted and reaches every **5 Enchant Levels**,\
an additional **random option** is granted.

* Only **one option** is applied.
* Higher Orb levels increase the **minimum–maximum values** of the option

| Orb type                                                                  | Enchant Option                                                    | Min \~Max |
| ------------------------------------------------------------------------- | ----------------------------------------------------------------- | --------- |
| <p>Orb of Wind <br>Orb of Darkness <br>Abyssal Orb <br>Orb of Clarity</p> | <p>STR </p><p>DEX </p><p>INT </p><p>CON </p><p>WIZ </p><p>CHA</p> | 1 \~ 1    |
| <p>Wind of Flame <br>Wings of Light <br>Burst Energy</p>                  | "                                                                 | 1\~2      |
| <p>Hand of the Demon <br>Wings of the Demon <br>Orb of Judgment</p>       | "                                                                 | 1\~3      |
| Rainbow                                                                   | "                                                                 | 2\~4      |
| Wrath of Water                                                            | "                                                                 | 2\~5      |

{% tab title="한국어" %}
### 💡 오브 스탯

오브가 생성될 때, 아래 조건에 따라 **스탯 옵션**이 적용됩니다.\
또한 오브를 인챈트할 경우, 인챈트 레벨에 따라 **추가 옵션**이 부여됩니다.

***

#### ◾ 오브 스탯 옵션 구성

오브에는 다음과 같은 스탯 옵션이 적용될 수 있습니다.

* [고정 옵션](orb-stats.md#id-1)
* [1차 스탯 옵션 (확률적)](orb-stats.md#id-2-1)
* [2차 스탯 옵션 (확률적)](orb-stats.md#id-3-2)
* [상성 옵션 (확률적)](orb-stats.md#id-4)
* [오브 인챈트에 따른 인챈트 옵션](orb-stats.md#id-5-1)

{% hint style="info" %}
⚠️ 옵션의 최소\~최대값은 밸런스 조정에 따라 일부 수치가 **비율 기준**으로 변경될 수 있습니다.
{% endhint %}

***

#### 1️⃣ 고정 옵션

모든 오브에 **100% 적용되는 기본 옵션**입니다.

* 각 오브에 정해진 최소\~최대 값 범위 내에서 랜덤 적용됩니다.
* 고정 옵션의 수치는 오브 종류에 따라 다르게 설정됩니다.

| 옵션        | 생성 확률 (%) |
| --------- | --------- |
| 회피율 (EVA) | 100       |
| HP        | 100       |

💡 고정 옵션 범위 안내\
👉 [장비별 적용 가능한 고정 스탯의 최소\~최대값을 확인할 수 있습니다.](orb-stats.md#fixed-option-range)

***

#### 2️⃣ 1차 스탯 옵션 (확률적)

오브 생성 시, **확률적으로 적용되는 스탯 옵션**입니다.

* 적용 확률은 오브마다 다릅니다.
* 정해진 최소\~최대 값 범위 내에서 랜덤 적용됩니다.
* 오브 레벨이 높을수록 옵션 생성 확률이 증가합니다.
* 각 옵션은 **독립 확률**로 적용됩니다.

| 옵션   | 생성 확률 (%) |
| ---- | --------- |
| STR  | 5\~8      |
| DEX  | 5\~8      |
| INT  | 5\~8      |
| CON  | 5\~8      |
| WIZ  | 5\~8      |
| CHA  | 5\~8      |

💡 1차 스탯 옵션 범위 안내\
👉 [1차 스탯 옵션의 최소\~최대값을 확인할 수 있습니다.](orb-stats.md#primary-stat-option-range)

***

#### 3️⃣ 2차 스탯 옵션 (확률적)

오브 생성 시, **추가로 확률 적용되는 스탯 옵션**입니다.

* 적용 확률은 오브마다 다릅니다.
* 정해진 최소\~최대 값 범위 내에서 랜덤 적용됩니다.
* 오브 레벨이 높을수록 옵션 생성 확률이 증가합니다.
* 각 옵션은 **독립 확률**로 적용됩니다.

| 옵션                          | 생성 확률 (%) |
| --------------------------- | --------- |
| 물리 데미지 (Phys. DMG)          | 8\~10     |
| HP                          | 8\~10     |
| MP                          | 8\~10     |
| 물리 치명타 데미지 (Phys. Crit DMG) | 8\~10     |
| 물리 치명타 확률 (Phys. Crit %)    | 3         |
| 마법 데미지 (Mag. DMG)           | 8\~10     |
| 마법 치명타 데미지 (Mag. Crit DMG)  | 8\~10     |
| 마법 치명타 확률 (Mag. Crit %)     | 3         |
| 재장전 시간 감소 (CDR)             | 4         |
| 회피율 (EVA)                   | 0.5\~1    |

💡 2차 스탯 옵션 범위 안내\
👉 [2차 스탯 옵션의 최소\~최대값을 확인할 수 있습니다.](orb-stats.md#secondary-stat-option-range)

***

#### 4️⃣ 상성 옵션 (확률적)

오브 생성 시, **속성 상성에 따른 부가 데미지 옵션**이 확률적으로 적용될 수 있습니다.

<figure><img src="../../../.gitbook/assets/image (1171).png" alt=""><figcaption></figcaption></figure>

* 적용 확률은 오브마다 다릅니다.
* 정해진 최소\~최대 값 범위 내에서 랜덤 적용됩니다.
* 오브 레벨이 높을수록 옵션 생성 확률이 증가합니다.

{% hint style="info" %}
상성에 따른 부가 데미지 옵션은 아래와 같이 해석됩니다.

<img src="../../../.gitbook/assets/image (621).png" alt="" data-size="original">\
이 표시의 의미는 '빛속성'을 가진 상대 (PC, NPC모두 포함)에게 2%의 부가 데미지를 입히는 것입니다. 만약 내가 적에게 입힐 최종 데미지가 200 이라면, 해당 적이 '빛속성'일 때, 최종 데미지가 204가 됩니다.

❓<mark style="color:blue;">**200\*(1+0.02)=204**</mark>
{% endhint %}

***

#### 5️⃣ 인챈트 옵션

오브를 인챈트하여 **인챈트 레벨이 5 단위에 도달할 때마다**, 추가적인 랜덤 옵션이 부여됩니다.

* 옵션은 **1종만 적용**됩니다.
* 오브 레벨이 높을수록 옵션의 최소\~최대 값이 증가합니다.

| 오브 종류                                          | 인챈트 옵션                                                            | 최소 \~ 최대 값 |
| ---------------------------------------------- | ----------------------------------------------------------------- | ---------- |
| <p>바람의 오브 <br>어둠의 오브 <br>심연의 오브 <br>청명의 오브</p> | <p>STR </p><p>DEX </p><p>INT </p><p>CON </p><p>WIZ </p><p>CHA</p> | 1 \~ 1     |
| <p>화염의 바람 <br>빛의 날개 <br>폭렬 에너지</p>             | "                                                                 | 1\~2       |
| <p>악마의 손 <br>악마의 날개 <br>심판의 오브</p>             | "                                                                 | 1\~3       |
| 레인보우                                           | "                                                                 | 2\~4       |
| 물의 분노                                          | "                                                                 | 2\~5       |

{% tab title="日本語" %}
### 💡 オーブステータス

オーブ生成時、以下の条件に基づいて ステータスオプションが適用されます。\
また、オーブをエンチャントすると、\
**エンチャントレベルに応じた追加オプション**が付与されます。

***

#### ◾ オーブステータスオプション構成

オーブには、以下のステータスオプションが付与される可能性があります。

* [固定オプション](orb-stats.md#id-1-opushon)
* [1次ステータスオプション（確率）](orb-stats.md#id-2-1suttasuopushon)
* [2次ステータスオプション（確率）](orb-stats.md#id-3-2suttasuopushon)
* [相性オプション（確率）](orb-stats.md#id-4-opushon)
* [エンチャントレベルに応じたエンチャントオプション](orb-stats.md#id-5-enchantoopushon)

{% hint style="info" %}
⚠️ オプションの最小値～最大値は、バランス調整により **割合基準** に変更される場合があります。
{% endhint %}

***

#### 1️⃣ 固定オプション

すべてのオーブに **100％適用** される基本オプションです。

* あらかじめ設定された **最小～最大値の範囲内** でランダム適用
* 固定オプションの数値は **オーブの種類ごと** に異なります

| オプション     | 生成確率 (%) |
| --------- | -------- |
| 回避率 (EVA) | 100      |
| HP        | 100      |

💡 **固定オプション範囲案内**\
👉 [装備別に適用可能な固定ステータスの最小～最大値を確認できます。](orb-stats.md#fixed-option-range)

***

#### 2️⃣ 1次ステータスオプション（確率）

オーブ生成時に、**確率的に適用**されるステータスオプションです。

* 適用確率はオーブごとに異なります
* 設定された **最小～最大値の範囲内** でランダム適用
* オーブレベルが高いほど、オプション生成確率が上昇します
* 各オプションは **独立した確率** で適用されます

| オプション | 生成確率 (%) |
| ----- | -------- |
| STR   | 5\~8     |
| DEX   | 5\~8     |
| INT   | 5\~8     |
| CON   | 5\~8     |
| WIZ   | 5\~8     |
| CHA   | 5\~8     |

💡 **1次ステータスオプション範囲案内**\
👉 [1次ステータスオプションの最小～最大値を確認できます。](orb-stats.md#primary-stat-option-range)

***

#### 3️⃣ 2次ステータスオプション（確率）

オーブ生成時に、追加로 **確率適用**されるステータスオプションです。

* 適用確率はオーブごとに異なります
* 設定された **最小～最大値の範囲内** でランダム適用
* オーブレベルが高いほど、オプション生成確率が上昇します
* 各オプションは **独立した確率** で適用されます

| オプション                    | 生成確率 (%) |
| ------------------------ | -------- |
| 物理DMG (Phys. DMG)        | 8\~10    |
| HP                       | 8\~10    |
| MP                       | 8\~10    |
| 物理クリDMG (Phys. Crit DMG) | 8\~10    |
| 物理クリ率 (Phys. Crit %)     | 3        |
| 魔法DMG (Mag. DMG)         | 8\~10    |
| 魔法クリDMG (Mag. Crit DMG)  | 8\~10    |
| 魔法クリ率 (Mag. Crit %)      | 3        |
| リロード時間短縮 (CDR)           | 4        |
| 回避率 (EVA)                | 0.5\~1   |

💡 **2次ステータスオプション範囲案内**\
👉 [2次ステータスオプションの最小～最大値を確認できます。](orb-stats.md#secondary-stat-option-range)

***

#### 4️⃣ 相性オプション（確率）

オーブ生成時、**属性相性に基づく追加ダメージオプション**が **確率的に付与される場合があります。**

<figure><img src="../../../.gitbook/assets/image (1171).png" alt=""><figcaption></figcaption></figure>

* 適用確率は、オーブごとに異なります。
* あらかじめ設定された **最小～最大値の範囲内** でランダムに適用されます。
* オーブレベルが高いほど、**オプション生成確率が上昇**します。

{% hint style="info" %}
相性に応じた追加ダメージオプションは以下のように解釈されます。

<img src="../../../.gitbook/assets/image (621).png" alt="" data-size="original">\
この表示の意味は、「光属性」を持つ相手（PC、NPC両方含む）に対して2%の追加ダメージを与えることです。もし私が敵に与える最終ダメージが200なら、その敵が「光属性」の場合、最終ダメージは204になります。

❓<mark style="color:blue;">**200\*(1+0.02)=204**</mark>
{% endhint %}

***

#### 5️⃣ エンチャントオプション

オーブをエンチャントし、エンチャントレベルが **5段階ごと** に到達すると、\
**追加のランダムオプション**が付与されます。

* オプションは**1種類のみ**適用されます。
* オーブレベルが高いほど、オプションの **最小～最大値が上昇** します

| オブタイプ                                        | エンチャントオプション                                                       | 最小～最大値 |
| -------------------------------------------- | ----------------------------------------------------------------- | ------ |
| <p>風のオーブ <br>闇のオーブ <br>深淵のオーブ <br>澄みのオーブ</p> | <p>STR </p><p>DEX </p><p>INT </p><p>CON </p><p>WIZ </p><p>CHA</p> | 1 \~ 1 |
| <p>炎の風 <br>光の翼 <br>爆裂エネルギー</p>               | "                                                                 | 1\~2   |
| <p>悪魔の手 <br>悪魔の翼 <br>審判のオーブ</p>              | "                                                                 | 1\~3   |
| レインボー                                        | "                                                                 | 2\~4   |
| 水の怒り                                         | "                                                                 | 2\~5   |
