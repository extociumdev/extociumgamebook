{% tab title="한국어" %}
#### ◾ 인챈트 가이드 바로가기

아래 가이드를 통해 필요한 인챈트 정보를 확인해 주세요.

***

**Weapon Enchantment** (무기 인챈트)

무기에 인챈트를 적용하는 방법을 안내합니다.\
인챈트 진입 방법, 화면 구성, 단계별 효과와 필요한 재료를 확인할 수 있으며,\
무기 인챈트 단계에 따라 발생하는 추가 부가 효과도 함께 설명합니다.

{% content-ref url="weapon-enchantment.md" %}
[weapon-enchantment.md](weapon-enchantment.md)
{% endcontent-ref %}

***

**Orb Enchantment** (오브 인챈트)

오브 전용 인챈트 시스템을 안내합니다.\
인챈트 진입 방법과 화면 설명, 단계별 효과 및 재료를 확인할 수 있으며,\
오브 인챈트 단계에 따른 부가 효과도 함께 확인할 수 있습니다.

{% content-ref url="orb-enchantment.md" %}
[orb-enchantment.md](orb-enchantment.md)
{% endcontent-ref %}

***

**Accessory Enchantment** (악세사리 인챈트)

악세사리에 인챈트를 적용하는 방법을 안내합니다.\
**현재 인챈트가 가능한 악세사리는 팔찌만 해당**되며, 인챈트 진입 방법과 화면 구성,\
단계별 효과와 재료를 확인할 수 있습니다.

{% content-ref url="accessory-enchant.md" %}
[accessory-enchant.md](accessory-enchant.md)
{% endcontent-ref %}

***

**Enchantment Bonus Effects** (인챈트 단계별 부가 효과)

인챈트 단계가 일정 수준에 도달하면 기본 효과 외에 추가적인 부가 효과가 발생합니다.\
각 단계별로 어떤 부가 효과가 적용되는지 자세히 확인할 수 있습니다.

{% content-ref url="enchant-stage-additional-effects.md" %}
[enchant-stage-additional-effects.md](enchant-stage-additional-effects.md)
{% endcontent-ref %}

***

**Enchantment Failed** (인챈트 실패 안내)

인챈트 실패로 장비가 파괴되었을 경우, 일정 확률로 특별한 아이템을 획득할 수 있습니다.\
이 페이지에서는 획득 가능한 아이템 종류와 확률 정보를 안내합니다.

{% content-ref url="enchantment-failed.md" %}
[enchantment-failed.md](enchantment-failed.md)
{% endcontent-ref %}

***

**Support Items** (인챈트 서포트 아이템)

인챈트를 도와주는 서포트 아이템을 안내합니다.

{% content-ref url="support-items.md" %}
[support-items.md](support-items.md)
{% endcontent-ref %}

***

#### 마지막 한 마디

> 인챈트는 강력한 시스템이지만\
> 충분한 이해 없이 진행하면 큰 손실이 발생할 수 있습니다.\
> 각 가이드를 확인한 후 신중하게 인챈트를 진행해 주세요.

**Orb Enchantment（オーブエンチャント）**

オーブ専用のエンチャントシステムをご案内します。\
エンチャントへの進入方法と画面説明、段階別の効果および素材を確認でき、\
オーブエンチャントの段階に応じた付加効果もあわせて確認できます。

{% content-ref url="orb-enchantment.md" %}
[orb-enchantment.md](orb-enchantment.md)

**Accessory Enchantment（アクセサリーエンチャント）**

{% content-ref url="accessory-enchant.md" %}

**Enchantment Bonus Effects（エンチャント段階別付加効果）**

エンチャント段階が一定水準に到達すると、基本効果に加えて追加の付加効果が発生します。\
段階ごとにどの付加効果が適用されるかを詳しく確認できます。

**Enchantment Failed（エンチャント失敗案内）**

エンチャント失敗により装備が破壊された場合、一定確率で特別なアイテムを獲得できます。\
このページでは、獲得可能なアイテムの種類と確率情報をご案内します。
