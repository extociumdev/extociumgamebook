### ◾ Equipment Mining Setup

Through **Equipment Set Effects**,\
you can increase your **daily gold mining limit** and gain **stat bonuses that help in combat**.

***

#### ◾ Basic Rules for Equipment Setup

* You can register **1 Weapon and 1 Orb** in the Mining Setup.
* Equipment registered in the Mining Setup is **used only for setup calculations**\
  and is **not related to direct equipment usage or combat participation**.
* The set effect starts from **Set No.1** with **Steel Sword** and **Abyssal Orb**,\
  and higher-level equipment applies **higher set numbers**.
* The applied set effect is determined by the **lower set level** of the two equipped items.
* **Example**
  * When equipping **Sword of Light + Water Wrath** → **Set No.10 effect** is applied.
  * When equipping **Awakening Darkness + Water Wrath** → **Set No.9 effect** is applied.

***

#### ◾ Set Effects

Set Effects applied through Equipment Setup are calculated based on the following rules:

{% hint style="info" %}
**Primary stat additional calculation:** Final primary stat value + n + n2 + n3...

**Secondary stat additional calculation:** Final secondary stat value \* (1 + (n% + n2% + n3%...)) )
{% endhint %}

### ◾ 장비 채굴 세팅

장비 세트 효과를 통해 일일 골드 채굴량 증가와 함께 전투에 도움이 되는 스탯 보너스를 획득할 수 있습니다.

***

#### ◾ 장비 세팅 기본 룰

* 채굴 세팅에 등록할 수 있는 장비는 **무기 1종, 오브 1종**입니다.
* 채굴 세팅에 등록된 장비는 **세팅 정보에만 사용되며**,\
  장비를 직접 사용하거나 **전투에 참여하는 것과는 무관합니다.**
* **‘강철검’과 ‘심연의 오브’**&#xBD80;터 **세트 1번**으로 적용되며,\
  장비의 레벨이 높아질수록 더 높은 **세트 번호**가 적용됩니다.
* 세트 효과는 장착한 두 장비 중 **세트 레벨이 더 낮은 장비를 기준**으로 결정됩니다.
* 예시
  * **‘빛의 검’ + ‘물의 분노’**&#xB97C; 장착할 경우 → **세트 10번 효과**가 적용됩니다.
  * **‘깨어나는 어둠’ + ‘물의 분노’**&#xB97C; 장착할 경우 → **세트 9번 효과**가 적용됩니다.

***

#### ◾ 세트 효과

장비 세팅으로 적용되는 세트 효과는 아래 기준에 따라 계산됩니다.

{% hint style="info" %}
❓**1차 스탯 추가 계산 :** 최종1차 스탯 값 + n + n2 + n3 ...

❓**2차 스탯 추가 계산 :** 최종 2차 스탯 값 \* ( 1 + ( n% + n2% + n3%...) )
{% endhint %}

### ◾ 装備採掘設定

装備セット効果を通じて、**1日のゴールド採掘量を増加**させることができ、\
あわせて**戦闘に役立つステータスボーナス**を獲得できます。

***

#### ◾ 装備設定の基本ルール

* 採掘設定に登録できる装備は **武器1種、オーブ1種**です。
* 採掘設定に登録された装備は **設定情報としてのみ使用**され、\
  装備を直接使用したり、**戦闘に参加することとは無関係**です。
* **「鋼鉄剣」と「深淵のオーブ」**&#x304B;ら **セット1番**として適用され、\
  装備のレベルが高くなるほど **より高いセット番号**が適用されます。
* セット効果は、装着している2つの装備のうち **セットレベルが低い方**を基準に決定されます。
* **例**
  * **「光の剣」＋「水の怒り」**&#x3092;装備した場合 → **セット10番効果**が適用されます。
  * **「目覚める闇」＋「水の怒り」**&#x3092;装備した場合 → **セット9番効果**が適用されます。

***

#### ◾ セット効果

装備設定によって適用される セット効果は、以下の基準で計算されます。

{% hint style="info" %}
**1次ステータス追加計算：**&#x6700;終1次ステータス値 + n + n2 + n3...

**2次ステータス追加計算：**&#x6700;終2次ステータス値 \* (1 + (n% + n2% + n3%...))
{% endhint %}
