# MetaMask transfer error (temporary solution: PC Chrome extension)

{% tabs %}
{% tab title="ENG" %}


If the MetaMask mobile app shows only \
&#xNAN;**\[Review]** and does not proceed when sending an NFT,\
send it via **MetaMask Chrome Extension on PC** as a workaround.<br>

* Install the **MetaMask Chrome extension** from the official website:
  * [https://metamask.io/ko/download](https://metamask.io/ko/download)

<figure><img src="../../../.gitbook/assets/image (2) (1).png" alt=""><figcaption></figcaption></figure>

* Click the fox icon (top-right in Chrome) → follow the steps to connect your wallet.
* Open the wallet → go to the **NFT tab** → select **BNB Chain**.

<figure><img src="../../../.gitbook/assets/image (4) (1).png" alt=""><figcaption></figcaption></figure>

* In the NFT tab, tap **⋮ (three dots)** → **+ Import NFT**.

<figure><img src="../../../.gitbook/assets/image (3) (1).png" alt=""><figcaption></figcaption></figure>

* Enter the following:
  * **EXTOCIUM NFT Contract**: `0xcbBb5398d976B056812F7F16F3EfF0b9DB8664b5`
  * **Token ID**: Check it in the in-game hero info and enter it.
* After import, the NFT becomes available to send.

### ⛔Confirm recipient wallet address (copy BNB Chain address)

* In the wallet app, tap the **Account name** (top-left on the main screen) →

<figure><img src="../../../.gitbook/assets/image (5) (1).png" alt=""><figcaption></figcaption></figure>

* select the **recipient wallet** →

<figure><img src="../../../.gitbook/assets/image (6) (1).png" alt=""><figcaption></figcaption></figure>

* tap **⋮** → **\[Address]** → copy the **BNB Chain address**.\
  (Icon: **two overlapping pages**)

<figure><img src="../../../.gitbook/assets/image (7) (1).png" alt=""><figcaption></figcaption></figure>

### Send

* Select the NFT → tap **\[Send]**
* Paste the copied address into the recipient field → tap **\[Continue]**
* &#x20;Normal state: **\[Continue]** is shown (not **\[Review]**)

<figure><img src="../../../.gitbook/assets/image (8) (1).png" alt=""><figcaption></figcaption></figure>

### ⛔Fee notice

<figure><img src="../../../.gitbook/assets/image (10).png" alt=""><figcaption></figcaption></figure>

* BNB is required in the wallet for gas fees.
* Recommended: **Hold at least 0.01 BNB** → check the fee → send.

{% tab title="日本語" %}


MetaMaskアプリでNFT送信時に **\[確認]（Review）ボタンのみ表示されて先に進まない**場合、\
\*\*PCのMetaMask（Chrome拡張）\*\*から送信すると回避できます。<br>

* 公式サイトから \*\*MetaMask（Chrome拡張）\*\*をインストールします：
  * [https://metamask.io/ko/download](https://metamask.io/ko/download)

<figure><img src="../../../.gitbook/assets/image (2) (1).png" alt=""><figcaption></figcaption></figure>

* Chrome右上の**キツネアイコン** → 案内に従って**ウォレット連携**します。
* ウォレットを開く → **NFT**タブ → **BNB Chain**を選択します。

<figure><img src="../../../.gitbook/assets/image (4) (1).png" alt=""><figcaption></figcaption></figure>

* NFTタブ右側の **⋮（3点）** → **+ NFTをインポート** を実行します。

<figure><img src="../../../.gitbook/assets/image (3) (1).png" alt=""><figcaption></figcaption></figure>

* 以下を入力します：
  * **EXTOCIUM NFT Contract**: `0xcbBb5398d976B056812F7F16F3EfF0b9DB8664b5`
  * **Token ID**: ゲーム内の英雄情報で確認して入力します。
* 登録後、送信可能になります。

### ⛔**受取ウォレットアドレス確認（BNB Chainアドレスをコピー）**

* ウォレットアプリのメイン画面左上の**Account名**をタップ →

<figure><img src="../../../.gitbook/assets/image (5) (1).png" alt=""><figcaption></figcaption></figure>

* 受取用ウォレットを選択 →

<figure><img src="../../../.gitbook/assets/image (6) (1).png" alt=""><figcaption></figcaption></figure>

* **⋮** → **\[アドレス]** → **BNB Chainのアドレス**をコピーします。\
  （アイコン：**重なった用紙2枚**）

<figure><img src="../../../.gitbook/assets/image (7) (1).png" alt=""><figcaption></figcaption></figure>

### 送信

* 送信するNFTを選択 → **\[送信]**
* 受取欄にコピーしたアドレスを貼り付け → **\[続行]**
* 正常：**\[確認]ではなく\[続行]が表示**

<figure><img src="../../../.gitbook/assets/image (8) (1).png" alt=""><figcaption></figcaption></figure>

### ⛔手数料注意

<figure><img src="../../../.gitbook/assets/image (10).png" alt=""><figcaption></figcaption></figure>

* 送信にはガス代として **BNB** が必要です。
* 推奨：**0.01 BNB以上**を保有 → 手数料確認 → 送信。
